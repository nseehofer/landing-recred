# RECRED - Landing Page

Landing page del pre-lanzamiento de **RECRED**, un buffet escolar inteligente que permite a alumnos, familias y kioscos gestionar las compras del recreo desde una sola app: sin filas, sin efectivo y con control de restricciones y presupuesto.

Proyecto realizado para la materia **TPI** de la **Tecnicatura en Desarrollo Web - UNLaM**.

## Descripcion

La landing comunica la propuesta de valor de RECRED a tres perfiles distintos (alumnos, padres y kiosqueros), explica como funciona en tres pasos y ofrece un formulario de acceso anticipado.

## Estructura del proyecto

```
landing-recred/
- index.html          # Estructura y contenido de la landing
- styles.css          # Estilos (mobile-first, responsive)
- assets/
  - landing.png       # Imagen de fondo del hero
  - logoHeader.png    # Logo de RECRED
  - instagram.png     # Icono de Instagram (footer)
- README.md
```

## Secciones

- **Hero** - Titulo principal, tagline y CTAs al formulario y a "Como funciona".
- **Beneficios** - Tres tarjetas segmentadas por perfil (alumno, padres, kiosquero).
- **Como funciona** - Tres pasos numerados que explican el flujo de uso.
- **Sumate** - Copy motivacional + formulario de registro para acceso anticipado.
- **Footer** - Marca, texto institucional y redes sociales.

## Tecnologias

- HTML5 semantico
- CSS3 (variables, grid, flexbox, media queries)
- JavaScript (validacion y envio del formulario)
- Tipografia: Montserrat

Abri `index.html` directamente en el navegador
